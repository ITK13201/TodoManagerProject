# TODO Manager

## Install

- docker
- docker-compose

## Versions

- docker: Docker version 20.10.7, build f0df350
- docker-compose: docker-compose version 1.29.0, build 07737305

## Usage

### Start container

```bash
docker-compose up -d
```

### Stop container

```bash
docker-compose down
```

### Logging

```bash
docker-compose logs -f
```
