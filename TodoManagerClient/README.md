# TODO Manager Client

## Install

- javac
- java

## Versions

- javac: javac 11.0.11

- java

```shell
openjdk 11.0.11 2021-04-20
OpenJDK Runtime Environment (build 11.0.11+9-Ubuntu-0ubuntu2.18.04)
OpenJDK 64-Bit Server VM (build 11.0.11+9-Ubuntu-0ubuntu2.18.04, mixed mode, sharing)
```

## Usage

### Build and Run

```shell
./exec.sh
```

### Build only

```shell
./exec.sh build
```

### Run only

```shell
./exec.sh run
```
